    <tr>
      <td>**runtime_config_overrides**</td>
      <td>Pre-Production</td>
      <td>
        A stringified key-value mapping of widgets to runtime configurations. When
        provided, 1FE will apply the runtime configurations when loading the widget
        instead of the widget's live runtime configuration.
      </td>
    </tr>
    <tr>
      <td>**widget_url_overrides**</td>
      <td>All Environments</td>
      <td>
        A stringified key-value mapping of widgets to bundle source. When
        provided, 1FE will use the provided bundle url instead of the widget's
        evergreen bundle.
      </td>
    </tr>
  </table>
</div>
