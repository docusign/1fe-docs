### `runtime_config_overrides`

A stringified key-value mapping of widgets to runtime configurations. When provided, 1fe will apply the runtime configurations when loading the widget instead of the widget's live runtime configuration.

**Supported Environments**: Pre-Production

### `widget_url_overrides`

A stringified key-value mapping of widgets to bundle source. When provided, 1fe will use the provided bundle url instead of the widget's evergreen bundle.

**Supported Environments**: All
